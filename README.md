## 介绍
- 模仿Google的[MVP架构](https://github.com/googlesamples/android-architecture/tree/todo-mvp)
- 模仿Bilibili安卓客户端的界面
- 主界面浏览，下拉刷新，分区刷新
- 分区界面按标签搜索，按发布时间、播放数等排序，自动加载更多
- 视频播放界面关注，收藏，评论，相关视频浏览
